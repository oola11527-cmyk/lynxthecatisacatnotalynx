HTML Cheat Sheet

What is HTML and What Can It Do?

HTML stands for Hyper Text Markup Language and is the standard markup language. It consists of a series of elements which tell the browser how to display the content such as paragraphs, headings, links, images e.t.c.

HTML is able to display text and images, organize content using headings and paragraphs, and is able to work with CSS and JavaScript. It can essentially create anything a website needs.

Links

There are a variety of links that can be used such as anchor links and external links. For anchor links to work, the location you want the link to jump to needs to be marked with the tag 

<h1 id="(The word you want it to jump to)">
and where you want to insert the link you must use an anchor tag
<a href="#(The word you want it to jump to)">

External links don't need a target section and only need an anchor tag to reach a different web page. Example of tag: 

<a href="(insert the URL)" target="_blank">Visit URL</a>
